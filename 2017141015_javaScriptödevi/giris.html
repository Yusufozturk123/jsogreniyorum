JavaScript Nedir?

JavaScript, yaygın olarak web tarayıcılarında kullanılan dinamik bir script dilidir. Etkileşimli ve dinamik web sayfaları oluşturmaya imkan sağlar. Yazım şekli ve ismi gereği Javaya benzemesinden dolayı sürekli karıştırılıyor olsalarda aslında JavaScript ve Java birbirinden farklı teknolojilerdir. Java nedir konusuna ilerleyen zamanlarda açıklık getireceğim.

JavaScript günümüzde gelişmiş olan tüm web sitelerinin neredeyse tamamında kullanılmaktadır. JavaScript web sitelerinin dinamik olmasını ve işlevselliğinin artmasını sağlar. Kullanıcı taraflı ve sunucu taraflı çalışabilir. Web sitelerinin kullanıcılar ile iletişim kurmasına yardımcı olur.

Örnek vermek gerekirse, bir web sitesinde form doldururken eksik veya hatalı giriş yaptığımızda uyarı veren veya bir girişe tepki olarak ekranda bulunan bir bilgiyi değiştiren JavaScripttir. Tabiki bu verdiğim örnekler JavaScriptin yapabildiklerinin binde biridir.

JavaScript dosyalarının uzantısı şeklindedir.

  • JavaScript Giriş
  • JavaScript Yazılım yapısı
  • Değişkenler
  • Operatörler
  • Aritmetik
  • Veri Tipleri
  • Fonksiyon Yapısı
  • Olaylar (Events)
  • Stringler, Sayılar, Diziler
  • If koşul yapıları
  • Döngüler
  • Tip Dönüşümleri
  • Çalışma yapısı ve genel mimarisi
